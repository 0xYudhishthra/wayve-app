### Installation

_We recommend installing Wayve by building from the source code as follows:_

```bash
# Download the source code
git clone https://github.com/wayve-ethglobal-ist/wayve-app
cd wayve-app

# Install dependencies
npm install

# Run app
npm run dev
```
